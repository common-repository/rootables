== Description ==

Root.abl.es is an expert community for programmers, developers and system administrators. Get feedback on your hacks, find answers to your tricky questions and learn to administer your Mac, Linux or Win system! 
This plugin allows you to share top hacks from <a href="http://root.abl.es?utm_source=wordpress&utm_medium=extend">root.abl.es</a> by tag, votes or views.
If you <a href="http://root.abl.es/accounts/register/?utm_source=wordpress&utm_medium=extend">create an account</a> you can even create your own personal shortlist of top tips! 




== Installation ==

1. Unzip the package and upload the rootables directory into your wp-content/plugins directory
1. Activate the plugin at the plugin administration page
1. Add the widget and they'll appear automagically!
1. (Optionally) change settings to choose which selection of hacks to show on your sidebar
